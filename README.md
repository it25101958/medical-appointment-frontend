# Medical Appointment Scheduling System 🏥

A professional full-stack monorepo featuring a **Spring Boot** REST API and a **Next.js** frontend. Designed for seamless medical appointment scheduling.

- **Frontend:** Next.js (App Router), Tailwind CSS, Axios
- **Database:** MySQL
- **Tools:** IntelliJ IDEA, VS Code, GitHub Projects
